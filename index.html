Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It supports procedural, object-oriented, and functional programming paradigms.

This documentation provides a guide to getting started with Python, its core features, and usage tips.

Getting Started
  1. Installation:
    • Windows:

      Download and run the installer from python.org, ensuring "Add Python to PATH" is checked.

    • macOS:

      Download and run the installer from python.org.

    • Linux:

      Install using the package manager:

      sudo apt update
sudo apt install python3
  2. Quick Start:
    1. Open your terminal or command prompt.
    2. Enter the Python shell by typing python or python3.
    3. Execute: 
      print("Hello, World!")
Basic Usage
  1. Basic Syntax:

    Python uses indentation for code blocks and comments with #:

    # This is a comment
print("Hello, World!")  # Output a message
  2. Data Types:
    • String: 
      name = "Joe"
    • Integer: 
      age = 20
    • List: 
      fruits = ["apple", "banana", "cherry"]
    • Tuple: 
      coordinates = (10.0, 20.0)
    • Dictionary: 
      person = {"name": "Alice", "age": 30}
  3. Control Flow:
    • If Statement: 
      if age > 18:
    print("Adult")
      else:
    print("Minor")
    • Loops: 
      for fruit in fruits:
    print(fruit)
      while age < 35:
    age += 1
  4. Functions:

    Define a function and call it:

    def greet(name):
    return f"Hello, {name}!"
print(greet("Alice"))
Modules and Packages
  1. Importing Modules:

    Import modules using:

    import os
import sys
  2. File Handling:

    Read a file:

    with open("file.txt", "r") as file:
    content = file.read()
    print(content)
  3. Third-party Packages:

    Install and use packages:

    • requests: 
      import requests
response = requests.get("https://api.example.com")
print(response.json())
    • numpy: 
      import numpy as np
array = np.array([1, 2, 3])
print(array)
Error Handling and Best Practices
  1. Error Handling:

    Handle exceptions using try-except:

    try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
  2. Best Practices:

    Follow PEP 8 guidelines for code style and write unit tests to ensure code reliability.

  3. Tips and Tricks:

    Use list comprehensions for concise code and context managers to handle resources efficiently.
